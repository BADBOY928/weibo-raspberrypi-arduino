weibo-raspberrypi-arduino
=========================

use raspberry pi to get real-time mentions(weibo), the mentions will be as the commands to control arduino.

Requires
----------
* requests
* pySerial

Usage
---------
1. Setting up configurations in config.py
2. python main.py
